**Requisitos Funcionais:**

1. Cadastro de produtos
2. Edição de produtos
3. Deleção de produtos
4. Cadastro de usuário
5. Login de usuário
6. Pagamento
7. Cadastro de cartão de débito ou crédito
8. Dashboard
9. Estoque
10. Favoritar produto
11. Carrinho
12. Lista de desejo
13. Notificação
14. Rastreamento de produtos
15. Parcelamento
16. Categorias
17. Previsão de entrega
18. Devolução
19. Cancelar envio do produto
20. Cancelar compra do produto



**Regras de Negócio:**

1. Código de vestimenta
2. Código de ética
3. Usuário precisa de login
4. Usuário precisa de cupons de desconto
5. Usuário precisa demonstrar interesse

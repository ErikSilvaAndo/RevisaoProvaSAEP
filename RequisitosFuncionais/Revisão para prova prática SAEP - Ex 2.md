**Exercício 2 -**



**Contexto:**

Uma cafeteria especializada em cafés gourmet está pensando em desenvolver um sistema para gerenciar pedidos online, controle de insumos, programa de fidelidade e avaliação dos produtos pelos clientes.



**Comandos:**

* Analise o contexto acima
* Liste pelo menos 13 requisitos funcionais para esse sistema
* Defina 3 possíveis regras de negócio para o funcionamento do programa de fidelidade ou das avaliações



\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_



###### **Respostas:**



**Requisitos funcionais:**

1. Envio de pedidos para os fornecedores;
2. Envio de notificações para o entregador;
3. Consulta de rota do entregador;
4. Controle de estoque;
5. Cadastro de produto;
6. Edição de produto;
7. Deleção de produto;
8. Cadastro de usuário;
9. Login de usuário;
10. Listagem de usuários;
11. Histórico de compras do usuário;
12. Avaliação de produtos;
13. Carrinho;

**Regras de Negócio:**

1. Geração de cupons, com base no tanto de compras do usuário;
2. Listagem de produtos mais comprados e melhores avaliados;
3. Histórico dos clientes que mais compram;
